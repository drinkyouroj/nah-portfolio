This directory is made to store GeoIP database. However, you will have to download this database yourself and put it in this directory.

Download GeoIP database from MaxMind website: 
http://dev.maxmind.com/geoip/geolite

* Choose [ GeoLite Country -> Binary / gzip ] *

You can also try to download from this direct link: 
http://geolite.maxmind.com/download/geoip/database/GeoLiteCountry/GeoIP.dat.gz

UnGzip GeoIP.dat.gz into this directory, so you'll have GeoIP.dat file here. That's it, your Instant Update will now also track country statistics!

Remember that this database is updated monthly so make sure you update it on your website regularly!
